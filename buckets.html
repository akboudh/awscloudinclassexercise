Create Two Buckets

Use separate buckets for input and output. Keep defaults (Block Public Access ON). Use the same Region.

Names

  • Input: notes-in-<yourinitials+3digits> (e.g., notes-in-ad123)
  • Output: notes-out-<yourinitials+3digits> (e.g., notes-out-ad123)

Steps

  1. S3 → Create bucket → make the input bucket → Create bucket.
  2. Repeat for the output bucket.
