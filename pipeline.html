Connect S3 Event to Lambda

We’ll configure the input bucket to trigger Lambda whenever a .md file is created.

Steps

  1. Open S3 → your input bucket (notes-in-…).
  2. Go to Properties → scroll to Event notifications.
  3. Click Create event notification and set:
FieldValue
Namemd-upload
Event typesAll object create events
Prefix(leave blank)
Suffix.md
DestinationLambda function → md_to_html_publisher

Save the event notification.
