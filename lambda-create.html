Create the Lambda Function

We will use the existing Academy role LabRole (no IAM creation).

  1. AWS Lambda → Create function
  2. Function name: md_to_html_publisher
  3. Runtime: Python 3.12
  4. Expand Change default execution role → select Use existing role → choose LabRole.
  5. Click Create function.

Set the handler

  1. Open the Code tab (top).
  2. On the bottm right-hand side, find the small card Runtime settings → Edit.
  3. Set Handler to lambda_function.handler → Save.
