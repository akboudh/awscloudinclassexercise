Overview

When a Markdown (.md) file is uploaded to an input S3 bucket, a Lambda function converts it to HTML and writes it to an output S3 bucket. The function also maintains a simple index.html in the output bucket.

What you’ll practice

  • S3 buckets (input/output)
  • Python Lambda authoring
  • S3 Event Notifications → Lambda
  • CloudWatch Logs for debugging

Environment constraints (AWS Academy)

  • No IAM role/policy creation. Use existing role: LabRole.
  • Keep everything in the same Region.
  • Website hosting may be blocked; download .html to view locally if needed.

Architecture

[You] → upload note.md ──▶  S3 (input: notes-in-…)
                               │ event: ObjectCreated (suffix .md)
                               ▼
                         Lambda (Python)
                               │  putObject: note.html, index.html
                               ▼
                         S3 (output: notes-out-…)
